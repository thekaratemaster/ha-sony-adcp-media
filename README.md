# ha-sony-adcp-media

Home Assistant custom integration for Sony projectors using ADCP over TCP.

## Contents

- `custom_components/sony_adcp_media/` - the integration
- `docs/integration-contract.md` - setup and automation-facing usage notes

## Installation

Copy `custom_components/sony_adcp_media/` into your Home Assistant `custom_components/` directory, restart Home Assistant, then add the integration from Settings -> Devices & Services.

## Notes

This repo is the source of truth for the Sony ADCP companion integration, branding, translations, and docs.
